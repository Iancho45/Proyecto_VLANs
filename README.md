# Proyecto_VLANs
Configuración de VLANs en Cisco Packet Tracer (segmentación de red, asignación de puertos, pruebas de conectividad)
Proyecto de práctica donde se realiza:

- Segmentación de red mediante VLANs
- Configuración de switches
- Asignación de puertos
- Pruebas de conectividad

Herramientas utilizadas:
- Cisco Packet Tracer
